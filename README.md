# 🏢 小区物业管理系统
一个基于 **Java + MySQL + JDBC** 开发的小区物业管理系统，实现住户信息管理、水电卡管理以及多种费用管理功能。本项目为课程设计实践项目，旨在通过完整系统开发流程提升数据库设计与Java开发能力。
## 📌 项目简介
随着小区规模不断扩大，物业管理涉及住户信息维护、水电缴费、供暖费用、物业费用等多项数据管理工作。传统人工登记方式效率低、易出错，且不利于数据统计。
本系统通过信息化手段，实现小区物业管理的规范化与自动化，适用于中小型小区的日常物业管理工作。
## 🛠 技术栈
* 开发语言：Java
* 数据库：MySQL
* 数据库连接：JDBC
* 开发工具：IntelliJ IDEA
* 数据库管理工具：Navicat
* JDK版本：1.8
## 🏗 系统架构
本系统采用分层架构设计：
* UI层（表现层）：负责界面交互
* Service层（业务逻辑层）：处理业务规则
* DAO层（数据访问层）：封装数据库操作
* Utils层：数据库连接工具类
通过分层设计实现低耦合，提高代码可维护性。
## 📂 数据库设计
系统主要包含以下数据表：
### 1️⃣ zrz_user
用于系统登录管理
* 用户ID
* 用户名
* 密码
* 用户角色
### 2️⃣ zrz_resident
住户信息表
* 住户ID
* 姓名
* 电话
* 房号
* 入住时间
### 3️⃣ zrz_property_fee
物业费用表
* 费用ID
* 住户ID
* 金额
* 缴费时间
* 缴费状态
### 4️⃣ zrz_heating_fee
供暖费用表
* 费用ID
* 住户ID
* 金额
* 缴费时间
### 5️⃣ zrz_water_card
水卡管理表
* 卡号
* 住户ID
* 余额
* 充值记录
### 6️⃣ zrz_electric_card
电卡管理表
* 卡号
* 住户ID
* 余额
* 充值记录
各数据表通过住户ID建立关联，实现费用与住户数据的对应关系。

## 🚀 项目运行步骤
1. 安装并启动 MySQL 数据库
2. 创建数据库（例如：`property_system`）
3. 执行建表 SQL 语句
4. 修改数据库连接配置（用户名、密码、数据库名）
5. 使用 IntelliJ IDEA 打开项目
6. 运行主程序启动系统
